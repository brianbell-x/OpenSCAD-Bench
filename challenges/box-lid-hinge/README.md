# Box Lid Hinge

## Success Criteria
- [ ] Fits 3mm wall thickness.
- [ ] Mechanism works (parts rotate relative to each other).
- [ ] Mounting holes are present and usable.
- [ ] Clearances are appropriate for 3D printing (usually >0.2mm).

## Skills Tested
- `assemblies` (multiple parts or print-in-place)
- `clearance` (tolerances for moving parts)
- `rotation` (aligning the hinge mechanism)
