# Headphone Hook

## Success Criteria
- [ ] Gap for desk is exactly 25mm (or slightly larger with tolerance).
- [ ] Hook extension is approximately 50mm.
- [ ] Headphone resting surface is curved/filleted.
- [ ] Geometry is manifold and printable on its side without supports (preferred).

## Skills Tested
- `boolean-ops` (difference for the clamp)
- `2d-extrusion` (often easier to draw in 2D and extrude)
- `fillets` (smoothing the hook)
